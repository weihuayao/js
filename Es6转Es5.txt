**webpack 使用 label将es6 转化为哦 es 5**	

1 首先安装 babel模块 

```bash
npm install -g babel-cli
```



2 然后创建一个文件夹，在文件夹里面初始化package.json文件

```bash
npm init -y //同npm init -yes
```

3然后在当前目录安装babel所需要的模块

```bash
npm install --save-dev babel-preset-es2015 babel-cl
```

4新建 babelrc

```
{
"presets":[
"es2015"
],
"plugins":[]
}
```

5当前的文件路径下运行

```bash
babel index.es6.js -o index.js
```
